# Kolla Ansible 文档

## Release 12.3.1.dev67

### 东软教育科技集团 IDC

### 2022年3月2日

---

Kollas的任务是为OpenStack云提供可生产的容器和部署工具

Kolla Ansible非常有个性,但也允许完全定制。这使得经验很少的操作人员可以快速部署OpenStack,随着经验的增长,可以修改OpenStack的配置,以满足运营商的准确需求
